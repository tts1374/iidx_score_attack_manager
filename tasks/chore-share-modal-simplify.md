# 共有モーダル簡素化 / 投稿文コピー文言変更

## 目的
- 「大会を共有（QR）」モーダルの情報量を減らし、主導線（画像確認 → 共有/保存/コピー）を明確化する。
- コピー操作のラベルを「投稿文をコピー」に変更し、コピー内容が投稿テンプレ全文であることを担保する。

## 非目的
- 共有画像の生成仕様（1080x1920 PNG）自体の変更。
- Web Share API / 画像保存 / クリップボード処理の実装方式変更。
- Single-tab 制御、Web Locks、Import/Export の仕様変更。

## 変更点
- 共有モーダル上部の説明アラート（`definition_only`）を削除。
- プレビュー領域の画像サイズ説明テキスト（`preview_image_size`）を削除。
- 共有セクションの見出し（`common.share`）を削除し、共有ボタンのみ表示。
- エクスポート内のコピー操作ラベルを「投稿文をコピー」に変更。
- 投稿文コピー内容を「投稿テンプレ全文（導入文 + ハッシュタグ + インポートURL）」に統一。
- 上記に伴う i18n 文言・テストを更新。

## 影響範囲（ユーザー / データ / 互換性）
- ユーザー影響:
  - 共有モーダルの表示項目とボタン文言が変わる。
  - クリップボードへコピーされる文面が URL 単体でなく投稿テンプレ全文になる。
- データ影響:
  - 永続化データ、Payload フォーマット、DB 変更なし。
- 互換性:
  - 既存の共有・画像保存・コピー操作APIは維持。
  - 生成画像フォーマット/サイズは変更しない。

## 実装方針（対象ファイル単位）
- `packages/web-app/src/pages/TournamentDetailPage.tsx`
  - 共有モーダルUIを簡素化（説明アラート削除、サイズ文言削除、共有見出し削除）。
  - 共有投稿文テンプレ文字列を `shareText` として生成。
- `packages/web-app/src/i18n/locales/ja.json`
  - `tournament_detail.action.copy_text` を「投稿文をコピー」に変更。
  - 投稿文テンプレ用キーを追加。
- `packages/web-app/src/i18n/locales/en.json`
  - `tournament_detail.action.copy_text` と投稿文テンプレ用キーを対応更新。
- `packages/web-app/src/i18n/locales/ko.json`
  - `tournament_detail.action.copy_text` と投稿文テンプレ用キーを対応更新。
- `packages/web-app/src/pages/TournamentDetailPage.test.tsx`
  - 共有モーダル表示要素とコピー内容の期待値を追加・更新。

## テスト観点
- 共有モーダルを開いたとき:
  - 説明アラートが存在しない。
  - 画像サイズ文言が存在しない。
  - 「共有」見出しが存在しない（共有ボタンは存在）。
  - 「投稿文をコピー」ボタンが表示される。
- 投稿文コピー時:
  - クリップボード書き込み値が「導入文 + ハッシュタグ + URL」の全文である。
- 既存動作:
  - Web Share API 実行と画像保存・コピー失敗時のフォールバックが回帰していない。

## ロールバック方針
- 単一コミットの `git revert` でUI/文言/テンプレ変更を一括復元可能にする。
- 予期しない回帰時は `TournamentDetailPage.tsx` と locale 変更を先に戻し、テスト差分を追従させる。

## Commit Plan（コミット分割計画）
1. 共有モーダルUI簡素化（説明/サイズ文言/共有見出しの削除）。
2. 投稿文コピー文言・テンプレ追加（TSX + i18n）。
3. テスト追加・更新と検証結果反映。
