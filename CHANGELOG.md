# CHANGELOG

## 1.2.0 - 2026-06-07

- 公開スコアタ一覧に、登録譜面の曲名・プレースタイル・難易度をプレビュー表示するようにしました。
- 端末に取り込み済みのスコアタを公開スコアタ一覧から除外するようにしました。
- 自分が公開したスコアタを公開一覧から削除できるようにしました。
- 画面内の「公開カタログ」表記を「公開スコアタ」に統一しました。

## 1.1.1 - 2026-05-17

- 本番公開カタログ API 設定を曲マスタ配布ワークフローでも保持するようにし、定期配布後も公開カタログが利用できる状態を維持しました。
- 公開カタログ一覧で開催中のスコアタが開始日経過後も表示されるよう、active 判定を終了日基準に修正しました。
- Service Worker が `cache: "no-store"` の公開カタログ API レスポンスを古いキャッシュから返さないようにしました。

## 1.1.0 - 2026-05-01

- 公開カタログ機能を追加し、大会の公開登録、一覧表示、詳細 payload 取得、ローカル公開状態管理に対応しました。
- 公開カタログのインポート導線、スタイル別件数表示、ページネーション、将来開催大会の絞り込みを改善しました。
- 公開カタログ API / Worker の環境設定、migration 実行、運用スクリプト、テストを整備しました。
- 公開カタログの削除失敗時や再登録時、公開処理中断時の復旧挙動を安定化しました。
- 曲マスタ更新頻度を週1回に変更しました。

## 1.0.0 - 2026-03-12

- アイコンを追加しました。
- 曲データの更新があるときにお知らせを表示するようにしました。

## 0.9.1 - 2026-03-04

- 共有まわりを改善し、QRシェアダイアログの視認性とシェアイメージ品質を向上しました。
- 大会作成フローを改善し、オーナー入力削除・ハッシュタグ補完・確認ステップ導線を最適化しました。
- ホーム/詳細画面のマイクロインタラクションとテーマ適用範囲を調整し、操作感を安定化しました。
